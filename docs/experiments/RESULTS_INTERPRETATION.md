# Results Interpretation

## What this benchmark shows

This v0.2.0 result set demonstrates that flux-sim now provides:
- explicit model-based metrics beyond structural counting,
- reproducible seed-controlled reports,
- internal synthetic class separation,
- comparison against both a structural baseline and an external baseline (Radon).

## What it does not show yet

This benchmark does not constitute external scientific validation against real-world failure corpora.
It is still an internal, synthetic, reproducible benchmark.

## Defensible claims after v0.2.0

- flux-sim implements an explicit effective model rather than only heuristic labels;
- flux-sim is reproducible under fixed seed and fixed benchmark corpus;
- flux-sim can be compared reproducibly against structural and Radon baselines;
- flux-sim should currently be described as a research-grade prototype, not a validated scientific instrument.
