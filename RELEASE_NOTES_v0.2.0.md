# flux-sim v0.2.0

## Highlights

- explicit effective model documentation
- discrete 2x2 Lindblad density evolution
- von Neumann entropy calculation
- effective relativistic factor
- explicit global constraint penalty
- reproducible seeded analysis
- schema and analysis versioning
- batch analysis
- synthetic benchmark corpus
- structural baseline
- external Radon baseline bootstrap

## Positioning

v0.2.0 should be presented as a reproducible research-grade prototype with explicit effective modeling and initial comparative benchmarking.

## Limitations

- synthetic benchmark only
- no real-world failure corpus yet
- no broad external validator comparison yet
- no claim of hardware-faithful quantum simulation
